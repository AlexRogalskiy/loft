Run this command to manually wake up a space:
```bash
loft wakeup [SPACE_NAME]   # optional flag: --cluster=[CLUSTER_NAME]
```

</TabItem>
<TabItem value="kubectl">

Run any kubectl command to wake up a sleeping space:
```bash
loft use space [SPACE_NAME]
kubectl get po
```
