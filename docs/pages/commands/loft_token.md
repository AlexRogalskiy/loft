Token prints the access token to a loft instance

## Synopsis


```
loft token [flags]
```

```
#######################################################
###################### loft token #####################
#######################################################
Prints an access token to a loft instance. This can
be used as an ExecAuthenticator for kubernetes

Example:
loft token
#######################################################
```


## Flags

```
  -h, --help   help for token
```
